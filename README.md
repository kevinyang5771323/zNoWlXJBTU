# 基于SpringBoot的民宿管理系统 java1008

## 项目概述

本项目是基于前后端分离的民宿管理系统，采用Java作为主要开发语言，后端基于SpringBoot框架，前端使用Vue3，数据库采用MySQL。

## 技术栈

- **开发语言**：Java
- **后端框架**：SpringBoot、MyBatis
- **前端框架**：Vue3
- **数据库**：MySQL
- **数据库管理工具**：Navicat
- **开发工具**：IDEA
- **JDK版本**：JDK8
- **构建工具**：Maven3+
- **推荐浏览器**：谷歌浏览器
- **系统运行环境**：IDEA、JDK8、MySQL5.7+、Navicat、Maven
